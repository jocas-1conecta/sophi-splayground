# Sophi's Playground — Cloudflare Pages

## Build Configuration
- **Build command**: `npm run build`
- **Build output directory**: `dist`
- **Root directory**: `/`
- **Node.js version**: `20`

## SPA Routing
This is a Single Page Application with client-side routing.
All routes should redirect to `/index.html`.

## Environment Variables
No server-side environment variables are needed.
All configuration is client-side (Supabase keys are public anon keys).
